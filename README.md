# Basketball Point Counter

## Project Overview

This project is a simple web-based basketball point counter for two teams: Home and Guest. The application allows users to increment each team’s score by 1, 2, or 3 points using interactive controls. It is built with standard front-end web technologies and runs entirely in the browser without any external dependencies.

## Features

- Separate score tracking for Home and Guest teams
- Buttons to add:
  - 1 point
  - 2 points
  - 3 points
- Real-time score updates
- Clean and minimal user interface

## Getting Started

### Prerequisites

To run this project locally, you need:

- A modern web browser (Chrome, Firefox, Safari, or similar)
- Optional: a code editor such as Visual Studio Code

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/halim-torayev/basketball-point-counter.git
   ```

2. Navigate to the project directory:

   ```bash
   cd basketball-point-counter
   ```

3. Open the `index.html` file in your browser.

## Usage

- Open the application in a browser.
- Use the +1, +2, or +3 buttons to increase the score for the Home or Guest team.
- Scores update immediately after each interaction.
