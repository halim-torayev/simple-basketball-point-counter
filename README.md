# Simple Basketball Point Counter

## Project Overview

This project is a simple web application that provides a basic basketball point counter for two teams: Home and Guest. The application allows users to increment the score of each team by 1, 2, or 3 points through interactive buttons. The interface is implemented using standard web technologies including HTML, CSS, and JavaScript.

## Features

- Display current scores for Home and Guest teams
- Buttons to add:
  - 1 point
  - 2 points
  - 3 points
- Interactive and responsive score updates in real time

## Technologies Used

- **HTML** — Structure of the application
- **CSS** — Presentation and layout of the interface
- **JavaScript** — Logic to update and display scores

## Getting Started

To run this project locally, you need:

- A modern web browser (e.g., Chrome, Firefox, Safari)
